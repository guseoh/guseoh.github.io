이번 글은 개인 블로그를 빠르게 여는 데 필요한 최소 절차를 기록한 메모입니다.

## 핵심 포인트

1. 정적 사이트 생성기(Astro)를 선택한다.
2. GitHub Actions로 빌드/배포를 자동화한다.
3. 글은 Markdown으로 작성하고 frontmatter 규칙을 고정한다.

## 다음 단계

태그/카테고리 기준으로 글 탐색 경험을 개선하고, 배포 후 피드백을 통해 디자인을 점진적으로 다듬을 예정입니다.
